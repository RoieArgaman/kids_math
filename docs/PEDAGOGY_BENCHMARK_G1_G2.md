## Scope

Benchmark of early-grade (Grades 1–2) math teaching practices across:
- Israel (MoE alignment baseline)
- Singapore
- Japan
- Finland
- United States
- China
- Germany

This doc focuses on **methods** that translate into app affordances: representations, task sequencing, routines, feedback, and mastery/spiral review.

## 1) Singapore — CPA + problem solving framework

**Core ideas**
- **C-P-A (Concrete → Pictorial → Abstract)** progression as a default teaching ramp.
- Problem-solving as the center, supported by: **Concepts, Skills, Processes, Attitudes, Metacognition** (Singapore MOE framework).

**What to import into kids_math**
- Treat `RepresentationType` (`concrete` / `pictorial` / `abstract`) as a **deliberate progression**, not just a label.
- Add “metacognition prompts” in feedback: “איזו דרך בחרת ולמה?” (lightweight; no long text).
- Use consistent diagram scaffolds (e.g., number line, part-part-whole / bar model when relevant).

## 2) Japan — Teaching Through Problem Solving (TTP)

**Core lesson structure**
- Pose a problem students don’t already have a procedure for.
- Independent solve time.
- Present & discuss multiple strategies (including incorrect ones when useful).
- Summarize/reflection.

**What to import into kids_math**
- Add “**strategy comparison moments**”:
  - After solving: show 2–3 canonical strategies and ask “איזו דרך דומה למה שעשית?”
  - For some items: a “which equation matches this story?” mini-check.
- Make the “share strategies” phase **visual** (number line jumps / decomposition), not paragraph-based.

## 3) Finland — playful inquiry + multimodal reasoning

**Core ideas**
- Reasoning via speech/gesture/artefacts; guided play; differentiation.

**What to import into kids_math**
- Short “**explore**” tasks (low stakes) before formal practice:
  - drag/arrange, quick estimation, “try 2 ways” prompts.
- Encourage multiple representations (tap-to-toggle between pictorial and abstract where feasible).

## 4) United States — Number Talks routine

**Core routine**
- 5–15 minute daily mental math discussion emphasizing multiple strategies and mathematical language.

**What to import into kids_math**
- A daily “**Number Talk**” block (3–5 prompts) with:
  - no time pressure
  - capture of the learner’s chosen strategy (optional)
  - focus strategies: make-10, doubles/near-doubles, counting-on

## 5) China — mastery + “teaching with variation” (Bianshi)

**Core idea**
- Systematic sequencing of tasks using **variation/invariance**:
  - “one problem, multiple changes”
  - “one problem, multiple solutions”

**What to import into kids_math**
- Replace “random-ish” practice with **micro-sequences**:
  - change one parameter at a time (number magnitude, unknown position, context)
  - keep invariant the target structure
- Use this to reduce misconceptions and build stable strategies.

## 6) Germany — Zahlvorstellung (robust number sense) + strategy ladders

**Core ideas**
- Build strong number meanings (ordinal/cardinal; part-whole) and move from counting to structure-based strategies.
- Emphasis on “make ten”, doubles, decompositions; structured representations (dice patterns, ten-frames).

**What to import into kids_math**
- Promote **part-whole** (`number-bonds`) as a first-class recurring skill (not only in early days).
- Default hints for Grade 1–2 addition/subtraction should suggest **structure** before recounting.

## Cross-cutting “best-of” that fits an app

These are the highest-signal techniques across countries that are app-friendly:
- **Representation progression (CPA)**: concrete/pictorial/abstract used intentionally.
- **Routines**: a predictable daily mental-math routine (Number Talk).
- **Strategy focus**: make-10, doubles, decompositions, number line jumps.
- **Variation sequencing**: change one thing at a time; avoid noisy drill.
- **Feedback as pedagogy**: hints that suggest a strategy + representation, not “try again”.
- **Metacognition lite**: short prompts that normalize multiple ways, without heavy writing load.
